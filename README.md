# POO TADS 26.1 - IFPI Campus Floriano

Este repositório contém exemplos de código desenvolvidos em sala de aula, bem como atividades resolvidas, referentes à disciplina de **Programação Orientada a Objetos (POO)** do curso de **Tecnologia em Análise e Desenvolvimento de Sistemas (TADS)** do **Instituto Federal do Piauí (IFPI) - Campus Floriano**.

O objetivo deste repositório é servir como material de apoio e registro de aprendizado prático da linguagem de programação Java e dos conceitos essenciais do paradigma orientado a objetos.

---

## 📂 Estrutura do Repositório

O repositório está subdividido em projetos/diretórios focados, organizados de acordo com os tópicos ministrados no decorrer da disciplina:

| Diretório | Descrição / Tópico Abordado | Principais Conceitos Exemplificados |
| :--- | :--- | :--- |
| [**`IntroPOO/`**](./IntroPOO) | Introdução à Programação Orientada a Objetos. | Classes, Objetos, Atributos, Métodos, Instanciação e Diagramas UML (Mermaid). |
| [**`encapsulamento/`**](./encapsulamento) | Fundamentos de Proteção de Dados e Modificadores de Acesso. | Modificadores `private`/`public`, Métodos Getters e Setters. |
| [**`AtividadeExtra/`**](./AtividadeExtra) | Atividade prática complementar de fixação. | Sobrecarga de Construtores, Manipulação de Arrays/Vetores e Laço For-Each. |
| [**`Atividades/Lista5/`**](./Atividades/Lista5) | Exercícios práticos referentes à 5ª lista de POO. | Encapsulamento Avançado, Regras de Negócio e Validações de Domínio. |

## 🚀 Como Executar os Projetos

Cada pasta listada acima representa um projeto Java independente estruturado para ser aberto diretamente no VS Code.

1. **Clonar o Repositório:**
   ```bash
   git clone https://github.com/profRonaldoIFPI/POO_TADS_26.1.git
   cd POO_TADS_26.1
   ```

2. **Abrir no VS Code:**
   Você pode abrir a pasta raiz ou individualmente cada projeto:
   ```bash
   code .
   ```

3. **Executar:**
   Abra a classe `App.java` correspondente ao projeto que deseja testar e clique em **Run** acima do método `main`, ou use o atalho padrão da IDE.
